# Lab 02 – Huawei Cloud Security Architecture

## Objective

Design a secure Huawei Cloud architecture for a fictional fintech company (PayCarry Türkiye), focusing on:

- DevSecOps
- Zero-CVE strategy
- PCI DSS continuous compliance
- Security governance
- Risk management

## Deliverables

- Assignment instructions (English)
- Assignment instructions (Turkish)
- Security assessment report
- Architecture diagram
- Risk register

## Technologies

- Huawei Cloud CodeArts
- CodeArts Inspector
- Host Security Service (HSS)
- SecMaster
- Web Application Firewall (WAF)
- Cloud Trace Service (CTS)
- Log Tank Service (LTS)
- Compliance Center
- Key Management Service (KMS)
