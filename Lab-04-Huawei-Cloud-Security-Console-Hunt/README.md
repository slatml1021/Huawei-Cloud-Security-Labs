# Lab 04 – Huawei Cloud Security Console & Documentation Hunt

## Objective

Locate and document 12 specific security, monitoring, compliance, and regional availability items across the Huawei Cloud console and official documentation.

For each item, the investigation records:

- The answer or verified fact
- The full Huawei Cloud console path or documentation URL
- A screenshot showing the source
- A short reflection on the investigation process

## Scope

The lab covers four main areas:

- Monitoring and logging
- Vulnerability management
- Compliance and baseline inspection
- Huawei Cloud Compliance Center and Trust Center

## Deliverables

- Answer sheet covering all 12 items
- Full console paths or official documentation URLs
- 12 tightly cropped screenshots
- Short reflection describing the most difficult item to locate

## Investigation Items

### Part A – Monitoring & Logging

1. Find a personal login trace in Cloud Trace Service
2. Identify the default CTS tracker name
3. Find the maximum free-tier CTS trace retention
4. Locate the LTS structured-parsing template for ELB

### Part B – Vulnerability Management & Compliance

5. Count the available HSS editions
6. Find an HSS emergency vulnerability entry
7. List and count all SecMaster compliance packs
8. Find the default scan schedule for Cloud Security Compliance Check 1.0

### Part C – Compliance Center & Trust Center

9. Locate the latest Huawei Cloud ISO 27001 certificate
10. Identify the regions covered by PCI DSS Level 1 certification
11. Locate the Huawei Cloud GDPR or KVKK position statement
12. Find a service available in TR-Istanbul but unavailable in the home region

## Huawei Cloud Services and Resources

- Cloud Trace Service (CTS)
- Log Tank Service (LTS)
- Host Security Service (HSS)
- SecMaster
- Compliance Center
- Trust Center
- Huawei Cloud regional availability documentation

## Evidence Format

Each answer should use the following structure:

1. Item number and title
2. Answer or verified fact
3. Console path or official URL
4. Screenshot reference
5. Additional notes, if necessary
